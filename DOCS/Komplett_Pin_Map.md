## Detailed Pin Assignments

###  Analog Inputs (ADC)

The project uses both ADC0 and ADC1 for analog readings.

#### üïπÔ∏è ADC0: Joystick (`ADC_JOY`)
-   **Joystick X-Axis:**
    -   BoosterPack Pin: `boosterpack.2`
    -   MCU Pin: `PA25`
    -   ADC Channel: `DL_ADC12_INPUT_CHAN_2`

-   **Joystick Y-Axis (Primary):**
    -   **Note:** The joystick Y-axis is read using `ADC1`, not `ADC0`. The `ADC_JOY` instance has a pin assigned for it (`boosterpack.6`), but the firmware reads the value from the `ADC_ACCEL` instance.
    -   See `ADC1` section for the pin used in practice.

####  accelerometer ADC1: Accelerometer & Joystick Y (`ADC_ACCEL`)
-   **Accelerometer X-Axis:**
    -   BoosterPack Pin: `boosterpack.23`
    -   MCU Pin: `PA17`
    -   ADC Channel: `DL_ADC12_INPUT_CHAN_6`

-   **Accelerometer Y-Axis:**
    -   BoosterPack Pin: `boosterpack.24`
    -   MCU Pin: `PA18`
    -   ADC Channel: `DL_ADC12_INPUT_CHAN_8`

-   **Accelerometer Z-Axis:**
    -   BoosterPack Pin: `boosterpack.25`
    -   MCU Pin: `PA21`
    -   ADC Channel: `DL_ADC12_INPUT_CHAN_5`

-   **Joystick Y-Axis (Actual):**
    -   BoosterPack Pin: `boosterpack.26`
    -   MCU Pin: `PA22`
    -   ADC Channel: `DL_ADC12_INPUT_CHAN_3`
    -   **Firmware Note:** `gSynthState.joy_y` is populated from this ADC reading.

### Audio & MIDI

#### üîä DAC12: Audio Output
-   **Function:** Generates the analog audio signal.
-   **BoosterPack Pin:** `boosterpack.30`
-   **MCU Pin:** `PA24`

#### üéπ UART0: MIDI Output (`UART_AUDIO`)
-   **Function:** Transmits MIDI data to the host PC.
-   **BoosterPack Pin:** `boosterpack.34` (TX)
-   **MCU Pin:** `PB2`
-   **Baud Rate:** 921600

### Digital I/O (GPIO)

#### üî≥ Buttons (`GPIO_BUTTONS`)
-   **Button S1:**
    -   BoosterPack Pin: `boosterpack.33`
    -   MCU Pin: `PA11`
-   **Button S2:**
    -   BoosterPack Pin: `boosterpack.32`
    -   MCU Pin: `PA12`
-   **Joystick Select:**
    -   BoosterPack Pin: `boosterpack.5`
    -   MCU Pin: `PA26`

#### üåà RGB LEDs (`GPIO_RGB`)
-   **Green LED:**
    -   BoosterPack Pin: `boosterpack.38`
    -   MCU Pin: `PB8`
-   **Blue LED:**
    -   BoosterPack Pin: `boosterpack.37`
    -   MCU Pin: `PB7`
-   **Note:** The Red LED is not configured in this project.

### üì∫ LCD Display

The LCD uses a combination of SPI for data and GPIO for control signals.

#### SPI1: LCD Data (`SPI_LCD`)
-   **Serial Clock (SCLK):**
    -   BoosterPack Pin: `boosterpack.7`
    -   MCU Pin: `PB16`
-   **Master Out Slave In (MOSI):**
    -   BoosterPack Pin: `boosterpack.15`
    -   MCU Pin: `PB15`
-   **Master In Slave Out (MISO):**
    -   BoosterPack Pin: `boosterpack.14`
    -   MCU Pin: `PA16`
    -   **Note:** MISO is configured but not actively used by the LCD driver.

#### GPIO: LCD Control
-   **Chip Select (CS):**
    -   BoosterPack Pin: `boosterpack.13`
    -   MCU Pin: `PA2`
    -   Instance: `LCD_CS_PIN`
-   **Data/Command (DC):**
    -   BoosterPack Pin: `boosterpack.31`
    -   MCU Pin: `PB14`
    -   Instance: `GPIO_LCD`
-   **Reset (RST):**
    -   BoosterPack Pin: `boosterpack.17`
    -   MCU Pin: `PB13`
    -   Instance: `GPIO_LCD`
-   **Backlight (BL):**
    -   BoosterPack Pin: `boosterpack.39`
    -   MCU Pin: `PB1`
    -   Instance: `LCD_BL`

### Other Peripherals

#### I2C1 (`I2C_1`)
-   **Function:** Configured but not actively used in the main application logic. Available for expansion.
-   **Serial Clock (SCL):**
    -   BoosterPack Pin: `boosterpack.9`
    -   MCU Pin: `PB3`
-   **Serial Data (SDA):**
    -   BoosterPack Pin: `boosterpack.10`
    -   MCU Pin: `PB4`

#### PWM (`PWM_AUDIO`)
-   **Function:** This PWM instance is configured on `boosterpack.40` but is **not used for audio output** in the current firmware. Audio is generated by the `DAC12`. This peripheral may have been intended for the buzzer on the EDU-MKII in a previous version.
-   **BoosterPack Pin:** `boosterpack.40`
-   **MCU Pin:** `PA12`
